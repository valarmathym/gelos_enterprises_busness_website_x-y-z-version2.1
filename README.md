
# Gelos Enterprises business Website – Version 2.1

##Live Demo:https://valarmathym.github.io/Gelos_Enterprises_Busness_Website_x-y-z-version2.1/

This project is an enhanced version of the Gelos Enterproses Business website developed as part of a web technology and innovation project. The website introduces modern interactive features focused on improving customer experience, accessibility, and engagement.

## Features Implemented

### AI Chatbot
- Integrated chatbot functionality to automatically respond to customer questions.
- Uses AI-based FAQ matching and automated response handling.
- Designed to improve customer support response time and user experience.

### Newsletter Signup System
- Developed a newsletter signup form using a JavaScript framework.
- Includes:
  - customer name validation
  - email validation
  - dynamic error handling
- Designed as a proof of concept for future JS framework adoption.

## Technology Evaluation
The project also includes:
- comparison of multiple technology solutions for each feature
- evaluation based on:
  - cost
  - compatibility
  - documentation
  - complexity
  - support
- recommendations for the most suitable implementation approach.

## Technologies Used
- HTML5
- CSS3
- JavaScript
- jQuery
- AI/Chatbot integration concepts
- Responsive Web Design

## Project Goals
- Improve customer interaction and accessibility
- Explore modern frontend technologies
- Demonstrate practical implementation of emerging web features
- Evaluate feasibility of integrating intelligent web technologies into eCommerce platforms
